# 🧭 Local Turístico — Conheça Busan

Projeto de página web estática desenvolvido com HTML e CSS, apresentando um guia turístico da cidade de **Busan (Coreia do Sul)**, com foco em atrações históricas e dicas de perfil de viagem.

## 📌 Sobre o projeto

A página mostra:
- Introdução ao destino turístico;
- Destaques culturais e históricos de Busan;
- 3 destinos imperdíveis com imagens;
- Indicações de público para cada atração (história, casais, famílias e orçamento).

## 🛠️ Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **Google Fonts** (Open Sans)

## 📁 Estrutura de pastas

```bash
Local turistico/

├── style.css
└── assets/
    ├── image.jpg
    ├── imagetemplo.jpg
    ├── imagetemplo2.jpg
    └── imageparque.jpg
```

## ▶️ Como executar

Como é um projeto estático, você pode abrir diretamente no navegador:

1. Abra a pasta do projeto.
2. Dê duplo clique no arquivo `index.html`.

Ou, se preferir no VS Code, utilize a extensão **Live Server** para visualizar com recarregamento automático.
